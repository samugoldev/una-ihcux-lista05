O que o programa faz

Recebe um valor em reais (R$)
Recebe a cotação do dólar
Mostra o valor convertido em dólares

Heurísticas aplicadas

Visibilidade do status: mostra mensagens como "Conectando..." e "Calculando..."
Prevenção de erros: trata entradas inválidas com try/catch
Estética: uso de cores e layout organizado no termina
